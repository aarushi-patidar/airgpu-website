# AirGPU

*AirGPU* is a decentralized compute-sharing platform that connects students and small developers who need GPU power with contributors who have idle hardware sitting unused. 

Downlod desktop app: https://github.com/SOMASEKAR17/AirGPU

Think of it as Airbnb for GPUs: instead of expensive cloud bills, users can tap into a community-driven pool of real machines to run AI training jobs, simulations, and research workloads at a fraction of the cost.

Contributors simply install a lightweight agent that shares their idle GPU or CPU cycles when their machine is not in use. 
Task submitters package their workloads into secure containers and submit them to the network, where AirGPU's scheduling system distributes them fairly across available contributors. Every task runs in an isolated environment, ensuring security and reproducibility without any setup complexity on the contributor's end.

Trust is built into the platform from the ground up. 
A verification layer establishes confidence between both parties before any execution begins, and a credit-based incentive system rewards contributors for their uptime and reliability. 
If a contributor goes offline mid-task, AirGPU automatically redistributes or checkpoints the work so no progress is lost. A real-time dashboard gives everyone visibility into resource usage, task status, and contribution history.

AirGPU turns wasted compute into a shared resource that democratizes access to the tools researchers and developers actually need, without the cloud price tag.
